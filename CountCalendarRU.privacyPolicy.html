Цель использования личной информации

Приложение просит вас отправить информацию о вашем адресе электронной почты, когда вы связываетесь с нами по электронной почте.
Адрес электронной почты, присланный клиентом, будет использоваться для ответов на вопросы и связи с клиентом для получения необходимой информации и не будет использоваться для каких-либо других целей.

Раскрытие личной информации третьим лицам

В этом приложении личная информация управляется надлежащим образом и не будет раскрыта третьим лицам, за исключением следующих случаев.
При наличии согласия заинтересованного лица.
Когда раскрытие информации необходимо для сотрудничества в соответствии с законами и нормативными актами.

Распространение рекламных объявлений

Данное приложение использует сторонние рекламные сервисы для публикации рекламы.
Для показа рекламы, которая подходит именно вам, может собираться и передаваться идентификатор вашего устройства (Advertising Identifier (IDFA)).
Для получения дополнительной информации, пожалуйста, обратитесь к следующему
https://support.apple.com/ru-ru/HT202074
Вы можете настроить свое устройство таким образом, чтобы вышеуказанные идентификаторы не использовались для создания профилей или показа персонализированных объявлений.
